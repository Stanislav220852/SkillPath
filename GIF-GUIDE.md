## Пошаговая инструкция

### Шаг 1: Запуск

```bash
npm run dev
```

Открой `http://localhost:5173` в браузере.

### Шаг 2: Настройки записи

Разрешение: **1280x720** (HD) — баланс качества и размера
FPS: **15-20** — достаточно для плавности, экономит место
Формат: **GIF**

### Шаг 3: Запись

#### GIF 1: Hero Demo (`hero-demo.gif`)

1. Открой главную страницу
2. Начни запись
3. Покажи:
   - Загрузку страницы (2 сек)
   - MouseSpotlight — движи мышкой по экрану (3 сек)
   - FloatingParticles — летающие частицы (2 сек)
   - Текст с TextScramble анимацией (2 сек)
   - Bento-секцию (пролистни вниз) (3 сек)
4. Останови запись
5. Обрежь до **8-10 секунд**

#### GIF 2: Quiz Demo (`quiz-demo.gif`)

1. Нажми "Начать тест" на главной
2. Начни запись
3. Пройди квиз:
   - Покажи выбор ответа (2 сек)
   - Следующий вопрос (2 сек)
   - Ещё вопрос (2 сек)
   - Результат теста (3 сек)
4. Останови запись
5. Обрежь до **8-10 секунд**

#### GIF 3: Roadmap Demo (`roadmap-demo.gif`)

1. Перейди на страницу роадмапов
2. Начни запись
3. Покажи:
   - Список роадмапов (2 сек)
   - Клик по роадмапу (1 сек)
   - Дерево навыков (2 сек)
   - Клик по навыку — открытие деталей (2 сек)
   - Прогресс-бар (2 сек)
4. Останови запись
5. Обрежь до **6-8 секунд**

#### GIF 4: Full Demo (`full-demo.gif`)

1. Начни с главной страницы
2. Начни запись
3. Быстро пролистни все секции:
   - Hero → Companies → Stats → Bento → Testimonials → FAQ (10-12 сек)
4. Останови запись
5. Обрежь до **12-15 секунд**

## Оптимизация GIF

### Через ezgif.com

1. Зайди на https://ezgif.com/optimize
2. Загрузи GIF
3. Настройки:
   - **Lossy GIF compression**: 30-40
   - **Colors**: 128 (если файл большой) или 256 (если норм)
   - **Resize**: 80% (если файл > 5MB)
4. Нажми "Optimize GIF"
5. Скачай результат

### Целевые размеры

| GIF | Макс. размер | Длительность |
|-----|-------------|--------------|
| hero-demo.gif | 3 MB | 8-10 сек |
| quiz-demo.gif | 4 MB | 8-10 сек |
| roadmap-demo.gif | 3 MB | 6-8 сек |

## Советы

### Движение мышки

- Двигай мышку **медленно и плавно**
- Покажи hover-эффекты на карточках
- Не води мышкой хаотично — это отвлекает

### Скорость

- Не торопись — зритель должен успеть прочитать
- Задерживайся на ключевых элементах **1-2 секунды**
- Не показывай всё слишком быстро

### Фон

- Убедись, что на экране нет лишних элементов (закрой другие вкладки)
- Используй **инкогнито-режим** чтобы не было расширений
- Отключи уведомления браузера

### Качество

- Не записывай в full-screen браузере — лучше в окне 1280x720
- Используй **стабильное соединение** чтобы страница грузилась быстро
- Предзагрузи страницу перед записью

## Частые ошибки

| Ошибка | Решение |
|--------|---------|
| GIF слишком большой (> 10MB) | Уменьши цвета до 128, обрежь до 8 сек |
| GIF дерганый | Увеличь FPS до 20, записывай плавнее |
| Не видно текст | Увеличь размер шрифта в браузере (Ctrl +) |
| GIF не загружается | Проверь формат, оптимизируй через ezgif |
| Плохое качество | Используй 1280x720, не сжимай слишком сильно |
