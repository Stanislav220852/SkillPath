# План: Вкладка «Чаты» с 3 подвкладками

## Цель
Добавить в навбар вкладку «Чаты» с тремя подразделами:
1. **Чаты с менторами** — существующий 1-on-1 чат, но как отдельная страница (не модалка)
2. **Общий чат** — публичный чат для всех пользователей
3. **Чаты пользователей** — личные сообщения между пользователями

---

## Что нужно сделать

### Бэкенд (FastAPI + SQLAlchemy)

#### 1. Новые модели данных
**Файл:** `src/app/skillpath-backend/src/models/chat_message.py` — расширить существующую модель

Добавить поля:
- `channel_type`: `Literal["mentor", "general", "direct"]` — тип чата
- `room_id`: `String` — ID комнаты (для direct: `user1_user2` отсортировано, для general: `"general"`, для mentor: `"mentor_{id}"`)

**Новая модель `DirectConversation`:** `src/app/skillpath-backend/src/models/direct_conversation.py`
- `id`, `user1_id` (FK), `user2_id` (FK), `last_message_at`, `created_at`
- Для отслеживания списка диалогов

#### 2. Новые API endpoints
**Файл:** `src/app/skillpath-backend/src/api/chat.py` — добавить:

- `GET /api/chat/general/history` — история общего чата
- `POST /api/chat/general` — отправить сообщение в общий чат
- `GET /api/chat/direct/conversations` — список диалогов пользователя
- `POST /api/chat/direct/{user_id}` — отправить ЛС пользователю
- `GET /api/chat/direct/{user_id}` — история ЛС с пользователем
- `WS /api/chat/ws/general` — WebSocket для общего чата
- `WS /api/chat/ws/direct/{user_id}` — WebSocket для ЛС

#### 3. Расширить сервис чата
**Файл:** `src/app/skillpath-backend/src/services/chat_service.py` — добавить:
- `save_general_message()` — сохранить сообщение в общий чат
- `get_general_history()` — получить историю общего чата (последние 100)
- `save_direct_message()` — сохранить ЛС
- `get_direct_history()` — история ЛС
- `get_user_conversations()` — список диалогов с последним сообщением
- `get_or_create_conversation()` — найти/создать диалог

#### 4. Расширить WebSocket Manager
**Файл:** `src/app/skillpath-backend/src/services/ws_manager.py` — добавить:
- `connect_general()` — подключение к общему чату
- `broadcast_general()` — отправить всем в общем чате
- `connect_direct()` — подключение к ЛС
- `send_to_direct()` — отправить собеседнику

#### 5. Миграция БД
Создать Alembic миграцию для:
- Добавления `channel_type` и `room_id` в `chat_messages`
- Создания таблицы `direct_conversations`
