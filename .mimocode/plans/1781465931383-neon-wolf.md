# Plan: Cinematic Guide Cat Presentation — Full Redesign

## Style: Dark Cinematic
- Dark background with neon accents (blue/purple glow)
- Cat runs between sections, particles float around
- Sections highlighted with neon glow + blur on non-active ones

## File to modify
`src/app/components/utils/GuideCharacter.tsx`

## What Changes

### 1. Delete old `CinematicOverlay` component
Remove the entire component (floating card at top, progress bar, step label). Replace with new system.

### 2. Create new `TourOverlay` component
**Purpose**: Visual effects during tour — blur + neon glow on active section
- **Blur**: All sections except active one get `backdrop-filter: blur(6px)` via CSS
- **Neon glow**: Active section gets pulsing blue/purple border with `box-shadow` animation
- **Floating particles**: Small glowing dots (blue/orange) float around the screen
- **No text, no progress, no cards** — just visual effects

### 3. Create new `TourBubble` component
**Purpose**: Speech bubble for cat's dialog
- Dark gradient background (`rgba(15,15,25,0.95)`)
- Neon border glow (blue/purple)
- Text appears **word by word** (not letter by letter) with 200ms delay per word
- Spring animation on mount (scale 0.8→1)
- Positioned above the cat character
- Close button (X) in corner

### 4. Enhance cat movement animations
**Running animation**: When cat moves between sections:
- Cat stretches horizontally (squash & stretch)
- Speed lines trail behind
- Cat position animates with `type: "spring", stiffness: 150, damping: 12`
- Dust particles spawn at cat's feet while running

**Arrival animation**:
- Cat arrives, squashes down (landing)
- Bounces up once
- Sits down (tail curls)
- Shows emoji above head with spring pop

**Idle at section**:
- Cat breathes (subtle y-axis oscillation)
- Tail sways slowly
- Blinks occasionally

### 5. Cat states during tour
| State | Animation |
|-------|-----------|
| `appear` | Fade in + scale from 0.5 to 1 (first step) |
| `running` | Squash stretch + horizontal movement + speed lines |
| `arriving` | Squash down + bounce + sit |
| `showItem` | Sits + emoji pops up with spring |
| `wave` | Waves paw (for greeting and farewell) |
| `excited` | Bounces up and down (for "Get Started") |
| `leaving` | Runs off screen to the right + fade out |
| `idle` | Breathing + tail sway + blinking |

- **Running animation**: 1.5-2 seconds between sections

- **Skip button**: Always visible in top-right corner
