# План: Исправление музыкального плеера — YouTube Lofi Girl

## Проблема

Текущий Web Audio API синтез звучит как 8-bit игрушка, а не как настоящий lofi. Пользователь хочет музыку как на Lofi Girl — мягкие джазовые аккорды, тёплый bass, vinyl crackle.

## Решение: YouTube IFrame Player API

Используем YouTube IFrame API для воспроизведения реального lofi стрима Lofi Girl.

### Источники музыки

| # | Название | YouTube ID | Описание |
|---|----------|-----------|----------|
| 1 | Lofi Girl Radio | `jfKfPfyJRdk` | Основной 24/7 lofi стрим |
| 2 | Chillhop Radio | `5qap5aO4i9A` | Chillhop Music стрим |
| 3 | The Jazz Hop Café | `gJ5aeR5U1Bg` | Jazz hop микс |

### Как работает

1. **YouTube IFrame API** загружается динамически (`script` тег)
2. Создаётся скрытый `<div id="ytplayer">` — YouTube плеер без UI
3. Play/pause через `player.playVideo()` / `player.pauseVideo()`
4. **Визуализатор** — CSS-анимированные столбцы (не привязаны к аудио, а пульсируют в ритм при isPlaying)
5. Смена трека — `player.loadVideoById(videoId)`

### MusicPlayer.tsx — новая структура

```typescript
// YouTube video IDs
const LOFI_TRACKS = [
  { id: "jfKfPfyJRdk", title: "Lofi Girl Radio", artist: "Lofi Girl" },
  { id: "5qap5aO4i9A", title: "Chillhop Radio", artist: "Chillhop Music" },
  { id: "gJ5aeR5U1Bg", title: "Jazz Hop Café", artist: "The Jazz Hop Café" },
];

// Загрузка YouTube IFrame API
function loadYouTubeAPI(): Promise<void> {
  return new Promise((resolve) => {
    if (window.YT) { resolve(); return; }
    const tag = document.createElement("script");
    tag.src = "https://www.youtube.com/iframe_api";
    document.head.appendChild(tag);
    window.onYouTubeIframeAPIReady = () => resolve();
  });
}

// В component:
// 1. При mount — загрузить API
// 2. При first play — создать YT.Player в скрытом div
// 3. Play/pause — player.playVideo() / player.pauseVideo()
// 4. Смена трека — player.loadVideoById()
// 5. Volume — player.setVolume(0-100)
```

### AudioVisualizer.tsx — CSS-only пульсация

Вместо Web Audio API анализатора — CSS-анимированные столбцы:
- 6 столбцов с `motion.div`
- При `isPlaying` — столбцы пульсируют в случайных высотах (simulated beat)
- При `!isPlaying` — столбцы на минимуме
- Используем `useEffect` с `setInterval` для рандомизации высот
